# Obsidian Recorder

Obsidian을 활용한 바이브 코딩 아카이빙 자동화 시스템입니다.

## 🚀 개요
이 프로젝트는 Antigravity와 옵시디언을 연동하여 작업 내용을 체계적으로 기록하기 위한 템플릿과 워크플로우를 제공합니다.

## 📁 주요 파일
- `Obsidian_Recorder_Master.md`: 프로젝트 전체 상태 및 히스토리 관리 마스터 대시보드.
- `Obsidian_Recorder_v1.0_Project_Initialization.md`: 초기 구축 및 템플릿 설계 로그.

---
## 📔 사용 방법
1.  **마스터 노트 생성**: 새로운 프로젝트 시작 시 `_Master.md` 형식의 노트를 생성합니다.
2.  **작업 아카이빙**: 작업 완료 후 Antigravity에게 "옵시디언 아카이빙해줘"라고 요청합니다.
3.  **자동 업데이트**: AI가 지정된 템플릿에 맞춰 버전 로그를 생성하고 마스터 노트를 업데이트합니다.
